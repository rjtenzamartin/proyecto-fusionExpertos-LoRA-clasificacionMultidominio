# proyecto-fusionExpertos-LoRA-clasificacionMultidominio
Proyecto que evalúa la Fusión de Expertos LoRA sobre BERT para clasificación de textos médicos y legales. Emplea DAPT mediante Masked Language Modeling para especializar adaptadores que luego se fusionan matemáticamente. Supera en F1-Score a modelos generalistas, destacando por su eficiencia (6.2 GB VRAM) y escalabilidad.
